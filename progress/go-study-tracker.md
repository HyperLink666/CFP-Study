## 学习技巧和建议

### ✅ 成功策略
1. **每天写代码** - 即使只有30分钟也要坚持
2. **先运行，后理解** - 看到效果再深入原理
3. **从小项目开始** - 不要一开始就做复杂项目
4. **阅读优秀代码** - GitHub 上的开源项目是最好的老师
5. **及时提问** - 遇到问题立即向导师询问
6. **做笔记** - 记录关键概念和常见陷阱

### ⚠️ 常见陷阱（提前知道）
1. **切片陷阱** - 共享底层数组导致的意外修改
2. **Goroutine 泄漏** - 忘记关闭 channel 或 context
3. **指针接收者 vs 值接收者** - 选择不当导致性能问题
4. **nil interface** - interface 的零值判断
5. **循环变量捕获** - 在 goroutine 中使用循环变量
