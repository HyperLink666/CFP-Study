## B. 基础语法（权重 15%）

**预计学习时间**: 第3-7天（5天）
**学习资源**: [A Tour of Go](https://go.dev/tour/), Go by Example

### ✅ 已掌握主题 (4/8)

- [x] **B.1 变量声明与数据类型** (2025-12-16) - **中等置信度**
  - 理解 Go 是静态类型语言（编译时确定类型）
  - 掌握三种变量声明方式：
    1. `var x int = 10` - 完整声明（最明确）
    2. `var x = 10` - 类型推断（简洁）
    3. `x := 10` - 短声明（最常用，仅限函数内）
  - 理解类型推断 ≠ 动态类型
  - 理解 Go 的类型确定后无法改变
  - **与 Python 对比**:
    - Python: `x = 10; x = "hello"` ✅ 可以改变类型
    - Go: `x := 10; x = "hello"` ❌ 编译错误
  - **关键区别**: Go 在编译时发现错误，Python 在运行时
  - **待实践**: 需要实际编写代码验证三种方式

- [x] **B.5 函数定义与调用** (2025-12-16) - **中等置信度**
  - 理解 `func` 关键字定义函数
  - 理解函数调用和执行顺序
  - 理解 Go 的代码组织最佳实践：逻辑在独立函数，main() 中调用
  - **与 Python 对比**: 语法相似 `def` vs `func`
  - **待学习**: 参数传递、多返回值、命名返回值等高级特性

- [x] **B.4.3 Range 遍历** (2025-12-21) - **高置信度** ✨
  - 理解 range 遍历三种用法：
    1. `for i, item := range slice` - 索引和值
    2. `for _, item := range slice` - 只要值（用 _ 占位）
    3. `for i := range slice` - 只要索引
  - 理解 Go map 遍历是**随机的**，Python 字典是有序的（3.7+）
  - **深入理解**：Unicode 字符导致的索引跳跃
    - 英文字符占 1 字节，中文占 3 字节
    - range 返回的索引是**字节索引**，不是字符索引
    - 例如：`"Hi世界"` 的索引是 `0 1 2 5`（不是 `0 1 2 3`）
  - **与 Python 对比**:
    - Python: for in/enumerate() 返回字符索引
    - Go: range 返回字节索引
  - **理解检查**: ✅ 两道题都答对，完全透彻理解

- [x] **B.4.2 Switch 语句** (2025-12-21) - **中高置信度** 🟡
  - 理解 switch 的基本用法和自动 break 特性
  - 理解 case 可以包含多个值：`case "A", "B":`
  - **独特特性**：无表达式 switch - 相当于优雅的 if/elif/else
    ```go
    switch {
    case score >= 90:
        fmt.Println("优秀")
    case score >= 80:
        fmt.Println("良好")
    }
    ```
  - 理解 switch 可以有初始化语句
  - **与 Python 对比**:
    - Python: 通常用 if/elif/else 或字典 + get
    - Go: switch 语法更清晰，自动 break 避免 bug
  - **待完成**: 理解检查题（3 道选择题）

## 学习技巧和建议

### ✅ 成功策略
1. **每天写代码** - 即使只有30分钟也要坚持
2. **先运行，后理解** - 看到效果再深入原理
3. **从小项目开始** - 不要一开始就做复杂项目
4. **阅读优秀代码** - GitHub 上的开源项目是最好的老师
5. **及时提问** - 遇到问题立即向导师询问
6. **做笔记** - 记录关键概念和常见陷阱

### ⚠️ 常见陷阱（提前知道）
1. **切片陷阱** - 共享底层数组导致的意外修改
2. **Goroutine 泄漏** - 忘记关闭 channel 或 context
3. **指针接收者 vs 值接收者** - 选择不当导致性能问题
4. **nil interface** - interface 的零值判断
5. **循环变量捕获** - 在 goroutine 中使用循环变量
