## 概念讲解

### 1. 项目技术栈分析

导师完成了以下分析：

**核心框架识别**：
- **CloudWeGo Hertz** - 字节跳动的高性能 HTTP 框架（替代常见的 Gin）
- **CloudWeGo Eino** - AI 编排框架
- **GORM** - ORM 框架（支持 MySQL + PostgreSQL）

**中间件与工具**：
- **Apollo 配置中心** - 携程开源的分布式配置管理
- **XXL-Job** - 分布式定时任务调度
- **Zap** - Uber 高性能日志库
- **go-redis/v9** - Redis 客户端
- **FastCache** - 内存缓存

**项目特色**：
1. **多数据源配置** - 支持 MySQL 和 PostgreSQL，主从数据库自动切换
2. **AI 对话** - 使用 WebSocket 实时通信
3. **链路追踪** - TraceID 全链路日志追踪
4. **复杂依赖** - 60+ 个直接依赖，包括各种云服务 SDK
